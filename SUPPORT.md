# Support

Forgebook issues are for actionable problems with this repository: broken recipe content, missing assets, site rendering bugs, metadata problems, or publishing workflow issues.

For broader Microsoft Foundry questions and discussion, use the Microsoft Foundry Developer Forum:

- Developer Forum: https://github.com/orgs/microsoft-foundry/discussions
- Get Help: https://github.com/orgs/microsoft-foundry/discussions/categories/get-help
- Product Feedback and Ideas: https://github.com/orgs/microsoft-foundry/discussions/categories/product-feedback-and-ideas
- Microsoft Foundry docs: https://learn.microsoft.com/azure/foundry/

If your question is about an existing Forgebook recipe, include the recipe URL, what you tried, and what happened.
