# Security

If you believe you have found a security vulnerability in Forgebook or one of its published recipes, please report it through Microsoft's coordinated vulnerability disclosure process rather than opening a public issue.

- Report vulnerabilities: https://msrc.microsoft.com/report
- Microsoft security guidance: https://www.microsoft.com/msrc

For non-security bugs in an existing recipe or the site, use the repository issue templates. For general Microsoft Foundry product questions, use the Microsoft Foundry Developer Forum.
